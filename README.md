# SmartRent mobile (tenant app)

Expo React Native client and FastAPI backend for the tenant experience. See `JUNIOR_DEVELOPER_GUIDE.md` for structure, env vars, and how to run locally.

Copy `frontend/.env.example` → `frontend/.env` and `backend/.env.example` → `backend/.env`, then adjust URLs and secrets.

After cloning, run `./scripts/enable-git-hooks.sh` once if you want the commit-msg hook that strips `Co-authored-by:` lines and IDE attribution footers.
